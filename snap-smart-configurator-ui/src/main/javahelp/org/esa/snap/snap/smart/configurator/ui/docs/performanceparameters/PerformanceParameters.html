Processing parameters

This preferences page provides options to customize processing parameters.

  • SNAP Values
    There are three processing parameters that may affect processing performance:
    • Tile size
      Defines the default tile size in pixels.
    • Cache size
      Defines the default cache size in MB.
    • Nb threads
      Sets the available number of threads.
  • Benchmark
    To help you to define faster processing parameters, you can launch benchmarks. You need to define a list of possible processing parameters for each values (tile size, cache size and nb threads). Each list must have at least one value, otherwise each values must be separated by a semi-colon (;). Then choose in the Proc. graph combobox the processing that will be used by the benchmark. Finally, press the button Compute to launch the processing dialog, sets processing parameters and Run. The benchmark will compute the processing with all given parameters and save the better result in SNAP Values.
